📊 Customer Churn Prediction AI 🚀
This project leverages Machine Learning to predict customer churn for subscription-based businesses. By analyzing historical data, the AI identifies patterns that lead to customer departure, allowing businesses to take proactive retention measures. 🏦✨

🌟 Key Features
🤖 Intelligent Modeling: Uses the Random Forest algorithm for robust and reliable classification.

📈 Insightful Visuals: Automatically generates a Confusion Matrix to track accuracy and a Feature Importance chart to reveal what drives churn.

💻 Interactive Interface: Features a real-time Terminal Predictor where you can input customer details and get instant risk assessments.

⚖️ Data Balanced: Implements preprocessing and scaling to ensure high-quality predictions.

⚙️ Installation & Setup
📥 Clone the repo:
Bash
git clone [https://github.com/gou-rab/Customer_Churn_Prediction.git](https://github.com/gou-rab/Customer_Churn_Prediction.git)

Bash

Bash
python customer_churn_prediction.py
